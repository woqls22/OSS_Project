# 실시간 이미지 스캔 페이지
## 주제
사용자가 이미지를 업로드하면, 실시간으로 스캔된 이미지의 형태를 반환하는 웹페이지[OpenCV라이브러리 이용]
- 원근 변환을 이용하여 기울어진 이미지도 정사각형 형태로 자동변환기능 지원
- 문서 OCR기능 지원
